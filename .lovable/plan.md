

# Stacioni i Librarisë — Faqja e Librave Online

A vintage-modern Albanian book e-commerce website with warm aesthetics, smooth animations, and a cozy bookstore atmosphere.

## Design System
- **Color palette**: Warm beige, dark brown, burgundy, forest green with paper texture backgrounds
- **Typography**: Elegant serif fonts (Playfair Display) for headings, clean sans-serif (Inter) for body text
- **Animations**: Fade-in on scroll, hover lift effects, page-turn transitions, floating book animations
- **Dark mode**: A warm "night reading" mode with deep tones

## Pages

### 1. Kreu (Home)
- Animated hero section with headline "Mirë se vini në Stacionin e Dijes" and CTA button
- Featured books carousel with hover lift animations
- Category preview cards (Romancë, Mister, Fantazi, Histori, Zhvillim Personal)
- Customer testimonials slider
- Newsletter subscription form
- Footer with social media links

### 2. Rreth Nesh (About Us)
- Storytelling section about the bookstore's inspiration
- Vintage-style animated timeline
- Mission & Vision cards
- Animated literary quote section

### 3. Librat (Books Catalog)
- Grid layout of book cards with filters (genre, author, price range)
- Smooth filtering animations
- Each card: cover image, title, author, price, "Shto në Shportë" button
- Hover effects: book lifts with deepening shadow

### 4. Detajet e Librit (Book Details)
- Large book image with subtle floating animation
- Full description, star ratings, and reviews
- "Shto në Shportë" button
- Related books suggestions section

### 5. Shporta (Shopping Cart)
- Clean minimal cart layout with animated quantity updates
- Item list with remove/update controls
- Running total price calculation
- "Vazhdo me Pagesën" checkout button

### 6. Kontakt (Contact)
- Elegant contact form with validation
- Embedded map placeholder
- Email & phone information
- Animated send button with confirmation feedback

## Shared Components
- Navigation bar with smooth scroll and active route highlighting
- Page transition animations (book page-turn feel)
- Dark mode toggle (vintage night reading mode)
- Responsive mobile-first layout with hamburger menu
- Sample book data (hardcoded catalog with Albanian titles and descriptions)

## Technical Notes
- All content in Albanian language
- Cart state managed with React context (localStorage persistence)
- No backend needed — uses hardcoded sample book data
- Fully responsive, optimized, and accessible
